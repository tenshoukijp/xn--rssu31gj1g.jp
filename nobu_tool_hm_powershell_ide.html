「PowerShell」と「開発環境」

  • 概要

    PowerShellの開発環境の紹介となります

  • Visual Studio

    Visual Studio は PowerShell編集の有力なエディタとなることでしょう。

  • Visual Studio Code

    Visual Studio Codeも PowerShell編集の有力なエディタとなることでしょう。
    拡張機能(Extension)としてPowerShell関連のものをインストールして利用してください。

  • PowerShell ISE

    Windows Management Frameworkに付いているPowerShell ISEも強力なエディタです。
    Visual Studio や Visual Studio Codeとは異なり、Power Shell 専用エディタとなりますので使う人を選びます。
