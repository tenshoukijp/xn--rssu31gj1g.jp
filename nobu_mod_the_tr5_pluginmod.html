太閤立志伝Ｖ 修正Mod for Windows 7以降

  • 概要

    太閤立志伝Ｖは、Windows Vista以降で、終了時にフリーズして窓が閉じないという問題があります。
    このModは、その対策Modです。

    フリーズ状態になっている場合、自動的に検知し、 ワーストケースでも「概ね9秒程度」のフリーズ状態で、
    自動的にゲームを強制終了します。

  • ダウンロード

    DOWNLOAD ⇒ TR5PluginMod.zipファイル。
    └更新日 %(year)04d/%(mon)02d/%(mday)02d

  • 動作環境:

    • Windows 7以降
    • .NET FrameWork 4.0 以降。

  • インストール

    上記ファイルをダウンロードして、
    下記の２つのファイルを「太閤立志伝Ｖフォルダ(taikou5.exeと同じフォルダ)」にコピーする。
    ファイルを置いておくだけで機能します。
    (太閤立志伝Vが起動する際に、自動的に読み込まれ機能します。)
    ・dsound.dll
    ・TermMod.exe

  • アンインストール

    コピーした２つのファイルを削除。
