列伝エディタ

  • 概要

    列伝を編集するためのツールとなります。

    以下の３つの項目を編集することが出来ます。

    • 武将列伝

      正式名称「戦国武将列伝」のことです

    • 家宝列伝

      正式名称「戦国名物事典」のことです

    • 城列伝

      正式名称「戦国名城物語」のことです

  • ダウンロード

    DOWNLOAD ⇒ RetsudenEditor.zipファイル。
    └更新日 %(year)04d/%(mon)02d/%(mday)02d

  • 動作環境

    • .NET FrameWork 4.0
    • Visual Studio 2013 Runtime をインストールしたことがない人は、インストールしてください。
      OSのビット数に関わらずx86版をインストールしてください。

  • インストール

    上記、.zipファイルを解凍し、RetsudenEditor.exe を 「天翔記フォルダ」に入れる。
    RetsudenEditor.exe を実行。

  • 使い方

    使い方の説明を要するするほどのUI要素はありません。

    従来の列伝エディタとは異なり、WYSIWYGに近くなっています。
