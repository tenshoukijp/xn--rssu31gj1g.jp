HmAllKill（秀丸エディタ・秀丸常駐秀丸エディタの強制終了）

  • 概要

    このプログラムは、実行中の全て「秀丸エディタ」「秀丸常駐エディタ」を全て一瞬で終了させます。

  • ダウンロード

    DOWNLOAD ⇒ HmAllKill.zipファイル。ver 1.01
    └更新日 %(year)04d/%(mon)02d/%(mday)02d

  • 動作環境

    • Microsoft .NET

      .NET Framework 4.0以上。

  • インストール

    • HmAllKill.exeを好きなディレクトリに配置してください。
    • なお、.zipファイルはブロック解除してください。
      (本ソフトには.exeが含まれていますので、原則的にはブロック解除をする必要があります)

  • 利用の手引き

    HmAllKill.exeを実行するだけで、「秀丸エディタ」「秀丸常駐エディタ」が全て終了します。

  • 利用用途の例

    • 秀丸エディタから、「秀丸エディタ本体自体を更新」するような「マクロ・バッチ」を組む際に必要。

      例えば、社内などにおいて秀丸を多人数で利用している場合、バージョンや環境を統一するのが一般的です。
      そのような場合に、「秀丸エディタ内からサーバー上に配置してある新しい秀丸エディタのプログラム群」を取得・更新するためには、
      「秀丸エディタのプロセス全て」を一発で全て終了させた後、秀丸エディタのプログラムを更新する必要があります。

    HmAllKillは、そのような自動更新のスクリプトやバッチを制作するシーンにおいて、役に立つことでしょう。
