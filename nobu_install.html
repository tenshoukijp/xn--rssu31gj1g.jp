天翔記のインストール先

  • Windows Vista, Windows 7, Windows 8/8.1, Windows 10

    Windows Vista, Windows 7, Windows 8/8.1, Windows 10 の場合、
    デフォルトのインストール先であるProgram Files(x86)は避けましょう。
    C:\usr\koei\tenshouki95 や C:\games\koei\tenshouki95 など、
    UACや空白スペースが無い、ご自身で用意したディレクトリ内にインストールしましょう。

    とにかく特別のユーザーアカウント制限(UAC)や書き換え制限が存在しない
    通常のディレクトリが強く望まれます。

    インストール先にユーザアカウント制限があるProgram Filesを選んでしまった場合、
    いくつかのツールは動作しません。
    (管理者権限で実行したとしても、実行できないものがあります。)

  • Windows XP

    Windows XPについては、
    C:\Program Files以下(即ちデフォルトのインストール先)ではなく、
    C:\usr\koei\tenshouki95 や C:\games\koei\tenshouki95 など、
    空白スペースが無いディレクトリ内にインストールしましょう。

  • Windows 95, Windows 98, Windows Me, Windows 2000

    当サイトの様々なプログラムは、これらの環境では、動作しません。
    少なくともXP以降(いくつかは7以降のOS)が想定されています。
