ヘックスマップエディタ

  • 概要

    「城マップ」や「野戦マップ」を編集するためのエディタとなります。

  • ダウンロード

    DOWNLOAD ⇒ HexMapEditor.zipファイル。
    └更新日 %(year)04d/%(mon)02d/%(mday)02d

  • 動作環境

    • .NET FrameWork 4.0
    • Visual Studio 2013 Runtime をインストールしたことがない人は、インストールしてください。
      OSのビット数に関わらずx86版をインストールしてください。

  • ヘックスマップエディタで作られた城データの例

    天翔記アップローダーに有志が作った城データがアップされています。

    • アップローダーの「.castle」
    • アップローダーの「城.」
    • 五稜郭

    「***.castle」といったファイルは、このヘックスマップエディタのうち、城マップエディタで読み込むことが出来ます。
    「***.castle」とは「１つの城のデータ」というだけであり、「どこの城」と決められたデータではありません。
    よって、どの城のデータとして読み込んでもかまいません、自由です。
