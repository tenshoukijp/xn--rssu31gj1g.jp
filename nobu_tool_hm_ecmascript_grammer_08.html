～文法メモ～ for...of

  • for...of と for...in

    for...of と for...in は、似て異なるもので、
    ・for...of は「values」を撮ってくる
    ・for...inは「keys(indexes)」を取ってくると考えれば良いかと思います。

    let data = [10,20,30]

for(let e of data) {
    hm.debuginfo(e);
}

// 結果 10,20,30

    従来から存在するfor...inは、

    let data = [10,20,30]

for(let e in data) {
    hm.debuginfo(e);
}

// 結果 0,1,2
