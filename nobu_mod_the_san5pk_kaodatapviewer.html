三国志V with PK用 kaodatap.s5 ビューワー

  • 概要

    三国志Ｖの顔グラとなる kaodatap.s5 の内容を「閲覧・出力・取込」するためのツールです。

  • ダウンロード

    DOWNLOAD ⇒ KaoDataPFaceViewer.zipファイル。
    └更新日 %(year)04d/%(mon)02d/%(mday)02d
    DOWNLOAD ⇒ ソースファイル。(プログラム書ける人用)
    └更新日 %(year)04d/%(mon)02d/%(mday)02d


  • 動作環境

    • Microsoft VC++ 再頒布可能パッケージ

      Visual Studio 2013 C++ ランタイム
      x86版をインストールしたことが無い方はインストールしてください。
      お手持ちのOSが64bitか32bitかには関わりなく、x86版のインストールが必要です。

    • Microsoft .NET

      .NET Framework 4.0以上。

  • インストール

    • KaoDataPFaceViewer.exeを「三国志Ｖのフォルダ(San5WPK.exeやSan5W95.exe)があるディレクトリに入れる。

    • KaoDataPFaceImporterも必要

      KaoDataPFaceImporter.exe が無い場合は、それも導入してください。

  • 使い方

    主に以下の３つの機能があります。

  • 閲覧

    普通に起動すると、閲覧状態となります。

    • 「→」キー、「↓」キー、「PageDown」

      次のページの顔グラ一覧を表示

    • 「←」キー、「↑」キー、「PageUp」

      前のページの顔グラ一覧を表示

    • 「home」キー

      最初のページの顔グラ一覧を表示

    • 「end」キー

      最後のページの顔グラ一覧を表示

  • 出力

    「o(オー)」のキーを押すと、顔グラ全部を出力出来ます。
    (output)の頭文字。

    • 「o」キー

      kaodatap.s5 の 顔グラを全部「kao_3bit」フォルダへと出力

  • 取込

    「i(アイ)」のキーを押すと、KaoDataPFaceImporter.exe を実行できます。

    • 「i」キー

      「kao_3bit」フォルダにあるxxxx.bmp を kaodatap.s5 へと 取込
